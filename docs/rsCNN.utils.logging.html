rsCNN.utils.logging module¶

rsCNN.utils.logging.MAX_BYTES = 104857600¶
rsCNN.utils.logging.get_root_logger(log_outfile=None)[source]¶

Get a root logger configured to log to stdout and optionally to a log file. Note that rsCNN log messages will also appear in the log files, so this is probably useful even if you don’t write your own logging messages.

Parameters

log_outfile (Optional[str]) – File to which logs should be written.

Return type

Logger

Returns

Logger configured for logging.

Example

>>> logger = get_root_logger('logfile.out')
>>> logger.setLevel('INFO')
>>> logger.info('This log message will be in logfile.out')
>>> logger.debug('This log message will not appear because the log level is set to info.')
